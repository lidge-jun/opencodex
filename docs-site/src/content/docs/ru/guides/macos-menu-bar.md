---
title: Строка меню macOS
description: Сборка и использование нативного приложения opencodex для состояния и управления прокси.
---

Приложение для macOS 12+ показывает в строке меню состояние, версию, время работы, PID, порт,
источник Bun runtime и состояние launchd. Там же можно запустить, перезапустить или остановить
прокси без терминала.

## Сборка и запуск

Сначала установите и настройте CLI `ocx`, затем выполните в репозитории:

```bash
bun run build:macos
open "dist/macos/OpenCodex.app"
```

При выбранном полном toolchain Xcode команда `UNIVERSAL=1 bun run build:macos` собирает arm64 и
x86_64 вместе; одних Command Line Tools достаточно только для текущей архитектуры. Результат получает
ad-hoc подпись для локального запуска; для распространения нужны Developer ID и notarization.

Приложение читает `ocx status --json` и делегирует запуск, перезапуск и остановку существующему CLI.
Перед перезапуском и остановкой показывается подтверждение, поскольку активные запросы могут быть
прерваны. Выход только из приложения строки меню не останавливает прокси.

Строка состояния прокси всегда отображается без затемнения. Версия, PID, порт, время работы, runtime,
состояние службы и сведения о CLI доступны в её подменю.

`ocx` автоматически ищется в PATH, Homebrew, Bun, Volta, pnpm, nvm и fnm. Если CLI не найден,
укажите исполняемый файл через **Выбрать ocx CLI…**. Для автозапуска добавьте собранное приложение в
**Системные настройки → Основные → Объекты входа**.
